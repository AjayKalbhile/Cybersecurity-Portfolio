## 🔍 Projects

### 🖥️ Project 1 — SOC Lab: Detecting Brute Force Attacks

> **Category:** Threat Detection & SIEM | **Difficulty:** Intermediate

**What I built:**
A home SOC lab ingesting 500+ log events/day from Windows and Linux endpoints. Configured Splunk with custom alert rules and dashboards. Detected 3 distinct brute-force attack patterns using log correlation across multiple sources.

**What I proved:** Real SIEM work isn't about the tool — it's about knowing what *normal* looks like so anomalies become obvious.

| Detail | Value |
|---|---|
| Tools Used | Splunk, Wazuh, Kali Linux, DVWA |
| Events Monitored | 500+ logs/day |
| Attacks Detected | 3 brute-force patterns |
| Output | Custom dashboards + alert rules |

### 🔓 Project 2 — Vulnerability Assessment Lab

> **Category:** Penetration Testing & Hardening | **Difficulty:** Intermediate

### 🎣 Project 3 — Phishing Simulation & Analysis

> **Category:** Social Engineering & Threat Intelligence | **Difficulty:** Intermediate

**What I built:**
Two complementary exercises:
1. **Phishing Analysis** — Investigated 10+ real phishing email samples. Extracted IOCs (malicious URLs, IPs, sender infrastructure) using VirusTotal and manual header analysis. Mapped TTPs to MITRE ATT&CK.
2. **Phishing Simulation** (Mastercard) — Designed a full phishing campaign, analyzed results by department vulnerability, and built a 5-slide security awareness training deck.

| Samples Analyzed | 10+ phishing emails |
